Model Architectures

Compare three CNN architectures. All output mean prediction + epistemic uncertainty via Gaussian NLL.

CUSTOM CNN
~8.2M params · From scratch
5× ConvBlocks with Squeeze-and-Excitation attention.
GAP + GMP fusion. Best for large custom datasets.
SpeedHigh
Accuracy (large data)Moderate
RESNET-50
~23.5M params · ImageNet pretrained
Fine-tune from layer2+. LayerNorm regression head.
Best accuracy for small/medium microscopy datasets.
SpeedMedium
AccuracyVery High
VGG-16
~138M params · ImageNet pretrained
Large receptive field ideal for texture-heavy images.
Strong for grain boundary / precipitate detection.
SpeedLow

Loss Function — Gaussian NLL
ℒ(μ, log σ², y) = ½ · [e−log σ²(y−μ)² + log σ²]
The model jointly optimises prediction accuracy (minimise residual) and calibrated uncertainty (learn σ per sample). When the model is uncertain it increases σ to avoid penalty — producing honest confidence intervals.

Example — cURL
# Predict fatigue life from a microscopy image curl -X POST "http://localhost:8000/predict?return_gradcam=true" \ -F "file=@steel_sample.png" \ -F "sample_id=SAMPLE_001"
Example — Python
import requests, base64, cv2, numpy as np # Send image to API with open("sample.png", "rb") as f: r = requests.post( "http://localhost:8000/predict", files={"file": f}, params={"return_gradcam": True} ) data = r.json() print(f"Risk: {data['risk_category']}") print(f"log(Nf): {data['log10_Nf']:.4f}") print(f"Nf: {data['Nf_cycles']:.2e} cycles") print(f"95% CI: [{data['log10_Nf_lower']:.3f}, {data['log10_Nf_upper']:.3f}]") # Decode annotated image img_bytes = base64.b64decode(data["annotated_image_b64"]) img_arr = np.frombuffer(img_bytes, np.uint8) img = cv2.imdecode(img_arr, cv2.IMREAD_COLOR) cv2.imshow("Result", img) cv2.waitKey(0)
